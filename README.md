# E-Commerce Sales & Customer Insights Dashboard

## 📊 Project Overview
This project analyzes an e-commerce dataset to uncover key business insights related to revenue, customer behavior, and product performance.

## 🔧 Tools Used
- SQL (Data Analysis)
- Power BI (Dashboard Visualization)
- Python (ETL & Data Processing)

## 📈 Key Insights
- Revenue is highly concentrated in a few regions
- Sales show strong seasonal trends
- Most customers are one-time buyers (low retention)
